# z8086 provenance

The base 8086 microarchitecture in this directory is derived from
[`nand2mario/z8086`](https://github.com/nand2mario/z8086) at commit
`596d9a6616b90c31f35b0d0947d4c1b11aa7a674` (retrieved 2026-07-14).

The upstream author releases the SystemVerilog implementation under the
Apache License 2.0. The upstream README separately cautions that the copyright
status of the reconstructed original Intel 8086 microcode is uncertain. This
project retains that notice. Batsugun-specific V25 extensions, peripherals,
clock-enable support, and bus adaptation are local changes.
